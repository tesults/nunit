--------------------------------------------------------------
Tesults
--------------------------------------------------------------

Tesults is a test automation results reporting service. https://www.tesults.com

Use the Tesults.NUnit extension to upload results to Tesults from NUnit. 

---------------------
Documentation
---------------------

View documentation at https://www.tesults.com/docs

Specifically: https://www.tesults.com/docs?doc=nunit3

Please note that NUnit will not recognize the extension without an .addins file. The documentation link provides complete details about this.
